## Overview

This repository contains the source code for the interactive, cinematic portfolio of Josh Franklin. Designed to behave like a premium operating system rather than a standard website, the interface leverages WebGL, custom shader environments, and generative choreography to create a digital universe that commands attention.

## Features

- **Immersive 3D Environments:** Powered by Three.js and custom GLSL shaders for realtime atmosphere.
- **Cinematic Motion Design:** Slow interpolation, environmental drift, and scroll-based scene evolution.
- **Premium User Experience:** Quiet by default, reactive when approached, and cinematic when the story demands impact.
- **High-Performance Architecture:** Optimized WebGL rendering wrapped in a modern, lightweight frontend structure.

## Technical Stack

- **Core:** HTML5, CSS3, JavaScript (ES Modules)
- **3D Graphics:** [Three.js](https://threejs.org/)
- **Shaders:** Custom GLSL 
- **Typography:** Google Fonts (Cormorant Garamond, Manrope)

## Running Locally

Because this project uses ES Modules (for Three.js imports), you must serve the files through a local HTTP server to avoid CORS issues. 

### Quick Start (Node.js)
If you have [Node.js](https://nodejs.org/) installed, you can easily serve the directory using `npx`:

```bash
npx serve .
```

### Quick Start (Python)
If you prefer Python, you can use the built-in HTTP server:

```bash
python -m http.server 8000
```
*Once the server is running, navigate to `http://localhost:3000` or `http://localhost:8000` (depending on your method) in your browser.*
